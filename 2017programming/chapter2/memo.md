## tmux

in terminal

`tmux` 起動
`tmux a` アタッチ（接続する）

in tmux

`Ctrl + b` -> `d` デタッチ＝離れる
`Ctrl + b` -> `c` ウィンドウを作成
`Ctrl + b` -> `数字` 数字のウィンドウ表示
`Ctrl + b` -> `p` 前の数字に移動
`Ctrl + b` -> `n` 後の数字に移動
`Ctrl + b` -> `&` ウィンドウを閉じる

`nc` TCP や UDP の読み書きを行う。
`telnet` リモートのコンピューターにアクセスし、ターミナルのセッションを開始する。(平文通信リスク)
